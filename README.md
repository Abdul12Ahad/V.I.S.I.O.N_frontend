# V.I.S.I.O.N

Visual Intelligent System for Interactive Observation and Navigation

## Features

- Image Upload
- Image Analysis using LLaVA
- Educational AI Explanations
- FastAPI Backend
- HTML/CSS/JS Frontend
- Ollama Local AI Integration

## Tech Stack

Frontend:
- HTML
- CSS
- JavaScript

Backend:
- FastAPI
- Python

AI Models:
- LLaVA 7B
- Mistral

## Run Backend

uvicorn app.main:app --reload

## Run Frontend

python -m http.server 3000
